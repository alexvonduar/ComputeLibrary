This kernel performs parallel reduction given an operation on x-axis.

Note
The data type must be passed at compile time using -DDATA_TYPE: e.g. -DDATA_TYPE=float
The operation we want to perform must be passed at compile time using -DOPERATION e.g. -DOPERATION=square_sum
The mean flag must be passed at compile time using -DMEAN if we want to compute the mean value
The width size must be passed at compile time using -DWIDTH e.g. -DWIDTH=128 if we want to compute the mean value
Parameters

[in]src_step_xsrc_stride_x * number of elements along X processed per workitem(in bytes)
[in]src_stride_yStride of the source tensor in Y dimension (in bytes)
[in]src_step_ysrc_stride_y * number of elements along Y processed per workitem(in bytes)
[in]src_offset_first_element_in_bytesThe offset of the first element in the source tensor
[in]partial_sum_ptrThe local buffer to hold sumed values. Supported data types: same as src_ptt
[in]partial_sum_stride_xStride of the output tensor in X dimension (in bytes)
[in]partial_sum_step_xpartial_sum_stride_x * number of elements along X processed per workitem(in bytes)
[in]partial_sum_stride_yStride of the output tensor in Y dimension (in bytes)
