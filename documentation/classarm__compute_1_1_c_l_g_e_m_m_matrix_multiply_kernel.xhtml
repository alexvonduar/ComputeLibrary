Detailed Description

OpenCL kernel to multiply two input matrices "A" and "B" .

All elements of the output matrix will be multiplied by alpha

Note
If the input tensors input0 and input1 have been reshaped respectively with CLGEMMInterleave4x4Kernel" and CLGEMMTranspose1xWKernel, the flag is_interleaved_transposed must be set to true
Attention
The second input tensor must have at least 2 dimensions (matrix)
